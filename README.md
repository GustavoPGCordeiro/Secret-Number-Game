# JavaScript Guess the Number Game

## Description

This project is a simple and interactive number guessing game built with JavaScript. The player must guess a randomly generated secret number between 1 and 25.

The game provides hints after each attempt, indicating whether the chosen number is higher or lower than the secret number. It also tracks the number of attempts and displays personalized feedback when the player wins.

## Features

### Random Secret Number

* Generates a random number between 1 and 25.

### Personalized Experience

* Asks the player for their name.
* Uses the player's name throughout the game.

### Hint System

* Indicates whether the guessed number is higher or lower than the secret number.

### Attempt Tracking

* Counts the number of guesses made by the player.

### Performance Feedback

* Displays different messages based on the number of attempts required to find the secret number.
